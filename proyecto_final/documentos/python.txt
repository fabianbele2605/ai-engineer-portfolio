Python is a high-level, interpreted programming language created by Guido van Rossum in 1991.
Python is known for its simple and readable syntax, making it ideal for beginners and experts.
Python supports multiple programming paradigms including procedural, object-oriented, and functional.
Python has a vast ecosystem of libraries like NumPy, Pandas, PyTorch and FastAPI.
Python is the most popular language for AI, Machine Learning and Data Science.
Virtual environments in Python allow isolating dependencies per project using venv or conda.
Python uses indentation to define code blocks instead of curly braces like other languages.
Python lists are ordered, mutable collections that can store elements of different types.
Python dictionaries store key-value pairs and are equivalent to JSON objects.
Python functions are defined with the def keyword and can have default parameter values.
Python classes support object-oriented programming with inheritance and encapsulation.
Python f-strings allow embedding expressions inside string literals using curly braces.
Python try-except blocks handle errors gracefully without crashing the application.
Python list comprehensions provide a concise way to create lists from existing iterables.
Python generators are memory-efficient iterators that yield values one at a time.
Python decorators are functions that modify the behavior of other functions.
NumPy provides fast mathematical operations on arrays and matrices for scientific computing.
Pandas provides DataFrame structures for data manipulation and analysis similar to Excel.
PyTorch is a deep learning framework used for building and training neural networks.
Scikit-learn provides simple and efficient tools for machine learning in Python.
FastAPI is a modern web framework for building APIs with automatic documentation.
Uvicorn is an ASGI server used to run FastAPI applications in production.
Pydantic provides data validation and settings management using Python type annotations.
SQLAlchemy is a SQL toolkit and ORM for database interactions in Python.
Requests is a popular HTTP library for making API calls in Python.
Pytest is the most popular testing framework for Python applications.
Python pip is the package installer for installing third-party libraries.
Python 3.12 introduced performance improvements and better error messages.
Python type hints allow specifying expected data types for better code documentation.
Python async/await enables writing asynchronous code for better performance with I/O operations.
Un entorno virtual en Python es un directorio aislado que contiene una instalación de Python y sus dependencias.
Los entornos virtuales permiten tener diferentes versiones de librerías para cada proyecto sin conflictos.
Se crea un entorno virtual con el comando python3 -m venv .venv y se activa con source .venv/bin/activate.
El entorno virtual activo se identifica por el prefijo (.venv) al inicio de la terminal.
El comando deactivate desactiva el entorno virtual y vuelve al Python del sistema.
Las listas en Python son colecciones ordenadas y mutables que pueden contener elementos de diferentes tipos.
Los diccionarios en Python almacenan pares clave-valor y son equivalentes a objetos JSON.
Las funciones en Python se definen con la palabra clave def y pueden tener parámetros con valores por defecto.
Las clases en Python permiten programación orientada a objetos con herencia y encapsulamiento.
El manejo de errores en Python usa bloques try-except para capturar excepciones sin que el programa falle.
